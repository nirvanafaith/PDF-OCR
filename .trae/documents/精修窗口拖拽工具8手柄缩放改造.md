# 精修窗口拖拽工具8手柄缩放改造

## 技术报告

### 需求

精修窗口中"拖拽"工具，点击文字时显示可拖拽的外框，支持：
- 拖动左右框变宽
- 拖动上下框变高
- 拖动边角自由拉伸
- 类似Word中对图片的操作体验

### 现状分析

当前 `MovableTextItem`（[refine_window.py:31-442](file:///c:/Users/E-VR/Documents/trae_projects/横校/ui/refine_window.py#L31-L442)）只有4个角手柄：

```
[topLeft]              [topRight]

[bottomLeft]           [bottomRight]
```

缺少4个边中点手柄，且没有光标样式变化。

### 目标设计

8个手柄布局（类似Word图片操作）：

```
[topLeft]----[top]----[topRight]
    |                     |
  [left]               [right]
    |                     |
[bottomLeft]-[bottom]-[bottomRight]
```

### 实现方案

#### 修改 1：`_create_handles` — 手柄从4个扩展到8个

将手柄名称列表从 `("topLeft", "topRight", "bottomLeft", "bottomRight")` 扩展为 `("topLeft", "top", "topRight", "left", "right", "bottomLeft", "bottom", "bottomRight")`。

#### 修改 2：`_position_handles` — 新增4个边中点手柄位置

边中点手柄位于矩形边的中点：
- `top`: 顶边中点
- `bottom`: 底边中点
- `left`: 左边中点
- `right`: 右边中点

#### 修改 3：`mouseMoveEvent` — 新增4个边中点手柄缩放逻辑

| 手柄 | 缩放行为 | 代码 |
|------|---------|------|
| top | 仅垂直缩放 | `new_rect.setTop(sr.top() + delta.y())` |
| bottom | 仅垂直缩放 | `new_rect.setBottom(sr.bottom() + delta.y())` |
| left | 仅水平缩放 | `new_rect.setLeft(sr.left() + delta.x())` |
| right | 仅水平缩放 | `new_rect.setRight(sr.right() + delta.x())` |

角手柄保持不变（同时影响两个维度）。

#### 修改 4：新增 `hoverMoveEvent` — 根据手柄位置设置光标

| 手柄 | 光标 | Qt枚举 |
|------|------|--------|
| topLeft, bottomRight | ↖↗ 对角箭头 | SizeFDiagCursor |
| topRight, bottomLeft | ↗↖ 反对角箭头 | SizeBDiagCursor |
| top, bottom | ↕ 垂直箭头 | SizeVerCursor |
| left, right | ↔ 水平箭头 | SizeHorCursor |
| 文字项内部 | ✥ 移动箭头 | SizeAllCursor |

#### 修改 5：新增 `hoverLeaveEvent` — 鼠标离开时恢复默认光标

### 不需要修改的部分

- `mousePressEvent` — `_handle_at` 方法已通用，自动支持新手柄
- `mouseReleaseEvent` — 不涉及具体手柄名称
- `_handle_at` — 遍历所有手柄，自动支持新手柄
- `_update_selection_visual` — 遍历所有手柄，自动支持新手柄
- `_sync_current_page` — 通过 rect() 和 pos() 获取数据，不受手柄数量影响
- `RefineWindow` 类 — 不需要修改

### 边界情况处理

1. **最小尺寸**：保持5像素限制，对所有8个手柄适用
2. **字体大小**：只跟高度关联，水平缩放不影响字体大小
3. **数据同步**：`_sync_current_page` 通过 rect()/pos() 获取位置，不受手柄影响
4. **Ctrl+滚轮缩放**：`update_zoom` 调用 `_position_handles`，自动支持8个手柄
