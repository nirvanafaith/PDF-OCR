# 横校（Horizontal Check）模块文档

## 一、整体定位与数据流

### 1.1 在四阶段流程中的位置

横校处于流程的**第三阶段**，接在纵校之后、精修之前。由 `MainWindow._setup_horizontal_stage` 创建，接收纵校产出的 `ocr_results` 和 `char_slices`，通过 `OCREngine.build_line_data` 方法转换为 `page_lines`（按页码组织的 `LineSlice` 列表）后传入。

### 1.2 输入数据规格

| 参数 | 类型 | 说明 |
|------|------|------|
| `page_lines` | `dict[int, list[LineSlice]]` | 键为页码，值为该页的 LineSlice 列表。每个 LineSlice 包含：`page_num`、`bbox`、`polygon`、`text`（整行文本）、`confidence`、`chars`（逐字符数据列表，每个元素含 `text`、`bbox`、`bbox_valid`）、`image`（行图像） |
| `page_images` | `list[PIL.Image]` | 索引对应页码，200 DPI 渲染的 PDF 页面 RGB 图像 |

### 1.3 输出数据规格

`finished_signal` 发射 `list[CorrectedLine]`，每个包含：`text`（校对后的行文本）、`bbox`（原始边界框）、`page_num`、`ignored`（是否被标记为忽略）。精修阶段接收横校的 `page_lines` 原始数据（含修改后的 chars）而非 `CorrectedLine`。

## 二、详细功能分解

### 2.1 字符级精细渲染引擎

横校不直接渲染整行文本，而是进行**逐字符独立渲染**：

- 遍历 `LineSlice.chars` 列表，对每个字符根据其独立 bbox 计算渲染位置和字体像素大小。
- **自适应字体缩放**：先用 bbox 高度计算像素字号，再用 `QFontMetrics.horizontalAdvance` 测量实际渲染宽度；若超出 bbox 宽度，按比例缩小字号，确保单字不溢出边界框。
- **居中定位**：水平和垂直方向均计算偏移量，使字符在其 bbox 内居中显示。
- **无字符数据时的回退**：若 `chars` 列表为空或所有字符 `bbox_valid` 为 False，则退化为整行渲染模式，用 `bbox[3] - bbox[1]` 计算行高作为字号。
- 字体硬编码为 `"Microsoft YaHei"`，抗锯齿渲染（`PreferAntialias`）。

### 2.2 鼠标悬停行切片预览机制

事件过滤器（`eventFilter`）监听左侧视图的鼠标移动事件：

- 当鼠标悬停在 `QGraphicsTextItem` 上时，从 `item.data(1)` 提取关联的 `LineSlice`。
- **切片图像生成**（`_make_slice_pixmap`）：从对应页面的原图中裁剪出该行区域——x 方向跨越整个页面宽度（从 0 到图片右边界），y 方向以行 bbox 为基准上下各扩展 20 像素的 padding。
- **双层缩放渲染**：切片图像先保持原始分辨率，再按当前 `zoom_level` 缩放到页面宽度匹配，保持图像长宽比。
- **预览定位**：预览图放置在场景坐标 `(0, line_y - preview_height)` 处，即行上方弹出，Z 值设为 100 确保在最顶层。
- **双级缓存加速**：
  - 单行缓存（`_slice_cache_id` / `_slice_cache_pixmap`）：避免同一行被反复裁剪和缩放。
  - 页面像素缓存（`_pixmap_cache`）：按 `(page_num, zoom_level)` 缓存缩放后的页面图像，切换页面或缩放时仅保留当前缓存。
- **右侧同步高亮**：在右侧 PDF 原图场景中用蓝色 2px 线宽的矩形（`QGraphicsRectItem`）标记该行位置。

### 2.3 右键上下文菜单

`_on_context_menu` 处理右键交互，仅当点击到 `QGraphicsTextItem` 时弹出：

- **修改文字**（`_on_modify_text`）：弹出 `QDialog`，内含 `QLineEdit` 预填当前文本，默认全选。确定后：
  1. 更新 `LineSlice.text` 为输入的新文本。
  2. 调用 `_sync_chars_with_text` 同步字符级数据：
     - 字符数相同时：逐个替换 `chars[i]["text"]`，保留原始 bbox。
     - 字符数不同时：按行 bbox 宽度等间距分配新字符的 bbox，标记 `bbox_valid: True`。
  3. 重绘当前页面。
  4. 记录修改操作到 `self.modifications` 列表。
- **忽略/删除**（`_on_ignore_line`）：设置 `LineSlice._ignored = True`，重绘后该行文字显示为灰色（`Qt.GlobalColor.gray`）。

### 2.4 页面导航与手势交互

- **按钮导航**：上一页/下一页/页码微调框跳转，翻页到边界时始终停在首末页。
- **鼠标滚轮翻页**：当滚动条已滚到顶部/底部且继续滚动时自动翻页（左右视图均支持），翻到最后一页的末尾或第一页顶端时停止。
- **Ctrl+滚轮缩放**：由 `zoom_utils.calculate_wheel_zoom` 统一处理，步长 0.1，范围 0.25~5.0。
- **滚轮翻页 vs 缩放优先级**：`QWheelEvent` 处理中，若 `Ctrl` 修饰键按下则缩放；否则检测滚轮方向与滚动条位置决定翻页。
- **手型拖拽**：`_on_hand_tool_toggle` 切换 `QGraphicsView.ScrollHandDrag`，光标改为 `OpenHandCursor`。

### 2.5 左右视图同步机制

- **三向信号绑定**：左右视图的垂直和水平滚动条的 `valueChanged` 信号互连，发出信号时阻塞对方信号（`blockSignals(True)`）避免循环触发。
- **双向翻页联动**：任一视图的滚轮翻页操作都会同步两个视图（单计时器设置两个滚动条位置）。
- **独立前景/后景**：左视图渲染文字叠加层（无背景图像），右视图仅渲染 PDF 原图，共享同一缩放级别。

### 2.6 自适应缩放与窗口事件

- **首次渲染**：`_first_render` 标志控制，100ms 延迟后自动调用 `_on_fit_height` 适应视图高度。
- **最大化检测**：`resizeEvent` 中检测 `WindowMaximized` 状态变化，首次最大化时自动适应高度。
- **适合宽度**：计算 `viewport.width() - 20` 与原始图像宽度的比值，设为新的 zoom_level。
- **适合高度**：类似逻辑，用视口高度除以图像高度。
- **缩放输入框**：支持输入百分比（如 `150%` 或 `150`），范围为 10%~1000%，回车确认，非法输入时恢复当前值。

### 2.7 完成横校流程

`_on_finish` 流程：

1. 统计 `modifications` 列表中 `modify_text` 和 `ignore` 操作的数量。
2. 弹出统计确认对话框，显示"修改文字：N 处；忽略行：M 处"。
3. 用户选择：
   - **确认**：调用 `_build_corrected_lines` 遍历所有页面、所有行，为每个 `LineSlice` 生成 `CorrectedLine`（含校对后文本和忽略状态），发射 `finished_signal`。
   - **返回继续校验**：关闭对话框，留在当前页面。
4. `back_signal`：返回纵校阶段（主窗口销毁横校窗口，恢复纵校窗口显示）。

## 三、详细 UI 布局

### 3.1 顶部工具栏（QToolBar）

从左到右布局，间距 6px，内边距 4px：

```
| ← 返回 | 分隔线 | 上一页 | 第N/M页 | [页码输入] | 下一页 | 分隔线 | 手型工具(可切换) | 放大 | [100%] | 缩小 | 适合宽度 | 适合高度 |
```

- `← 返回`：发射 `back_signal`。
- 页码输入：`QSpinBox`，范围 1~total_pages，支持键盘上下键微调。
- 手型工具：`QPushButton`，`setCheckable(True)`，选中状态 `checked` 时切换拖拽模式。
- 缩放输入：`QLineEdit`，固定宽 55px，文字居中，仅接受回车确认。

### 3.2 中部主区域（左右双视图）

- **分配比例**：`QHBoxLayout`，间距 2px，左右各占 1 份拉伸因子（`addWidget(view, 1)`）。
- **左侧视图**（主交互区）：
  - 无背景图像，直接渲染 `QGraphicsTextItem` 文本图元。
  - 场景背景为白色（`setBackgroundBrush(Qt.GlobalColor.white)`）。
  - 右键菜单策略 `CustomContextMenu` → 自定义菜单。
  - 鼠标追踪开启（`setMouseTracking(True)`）实现悬停预览。
- **右侧视图**（参考区）：
  - 纯 `QGraphicsPixmapItem` 显示缩放后的 PDF 原图。
  - 无交互菜单，仅用于视觉参考。

### 3.3 底部布局

```
| (拉伸) | 完成横校 | (拉伸) |
```

仅一个按钮，点击后弹出确认对话框。

### 3.4 模态对话框

**修改文字对话框**：
- 窗口标题："修改文字"
- 最小宽度：400px
- 组件：提示标签 + `QLineEdit`（预填原文本，全选）+ 确定/取消按钮水平布局
- 回车键绑定接受（`line_edit.returnPressed.connect(dialog.accept)`）

**确认完成对话框**：
- 窗口标题："确认完成横校"
- 显示统计信息（QLabel）
- 按钮："确认"（接受）/ "返回继续校验"（拒绝）
